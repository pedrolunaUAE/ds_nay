
# Información Sociodemográfica de Nayarit 2020 — Publicado (datos integrados)

Este paquete incluye el front completo y tu `data/nayaritData.js` ya integrado.

## Ejecutar sin Docker
```bash
python3 -m http.server 8080
# http://localhost:8080
```

## Ejecutar con Docker
```bash
docker compose up --build -d
# http://localhost:8080
```

## Notas
- Coloca el logo institucional en `img/LogotipoLealtad.png` si deseas personalizar el encabezado del PDF.
- El botón **Descargar PDF** captura todas las secciones en un PDF continuo.
- Tooltips y etiquetas muestran **%** y **(absoluto)**.
